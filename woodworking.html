Woodworking


CNC Router


The Machine

We used a DeWalt router and the axes move using a stepper motor moving on V-Slots. The design was inspired by the Ooznest Ox.

Assembly

The assembly was pretty simple. The design uses V-Slots, laser cut plates, wheels, and T-Nuts.

Electrical Box

At the beginning, we thought the electrical box was the problem. The drives were overheating even if we were using a fan, so we though the box was overcrowded. It turned out that we needed a big fan even when the box was open which doesn't make sense, so the cheap drives we bought were the problem. We bypassed the problem for a while thanks to the cold winter in Québec.

Machining

The machine as a 1 m by 1.5 m cutting area. We use a sacrificial MDF board to be able to screw or nail the wood down and to machine all the way through the part.

Vacuum System

The wood dust was a big problem, so I added this vacuum system. It works perfectly and the machine even built it itself!

Planing a Timber

This was a crazy project. One of our friends wanted to plane a timber, but didn't have a planer. It took a while, but we managed to plane it!

Table Saw

Fun fact, this table saw can detect if a finger is touching the blade (by conductivity) and stops immediatly if it does. It breaks the blade, but your finger is all right! That also means that if you need to saw a piece of wood with nails (conductive) in it, you have to bypass this feature.
